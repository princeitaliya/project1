Css Color

            RBG
            color = 0 to 255
            RBG  = red ,blue,green 
            rgb(0,0,0); = black
            rgb(255, 255, 255); = white
            alpha =  0.0 fully transparent and 1.0 not transparent

            HEX  = hexadecimal 
            rr(red), bb(blue), gg(green)
            #000000 = black
            #fffffff = white
            alpha =  0.0 fully transparent and 1.0 not transparent

            HSL
            H = Hua. degree 0 to 360 = 0 is red , 120 is green, 240 is blue
            S = Saturation 0% is gray and 100% full Color
            L = Lightness  0% is black and 100% is white
            alpha =  0.0 fully transparent and 1.0 not transparent
